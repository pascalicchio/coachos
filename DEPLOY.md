# Deploy Instructions

## Option 1: Netlify (Recommended - FREE)

1. Go to https://app.netlify.com/drop
2. Drag and drop the `public/` folder
3. Your site is live!
4. Add custom domain in settings (optional)

## Option 2: Vercel (Also Free)

1. Go to https://vercel.com/dashboard
2. Import from GitHub
3. Deploy!

## Form Setup (For Waiting List)

1. Go to https://formspree.io
2. Create free account
3. Create new form
4. Replace `YOUR_FORM_ID` in signup.html with your form ID
5. You'll get email notifications when people sign up!

## Custom Domain

After deploying:
- Netlify: Settings → Domain Management → Add domain
- Point your domain's DNS to Netlify

## Current Files

- `public/index.html` - Main landing page
- `public/signup.html` - Waiting list signup
