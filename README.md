# CoachOS - Gym Management Simplified

## The Vision
The simplest gym management software for BJJ/martial arts gym owners.

## Current Status
- [x] Landing page created
- [ ] Laravel setup
- [ ] Database migrations
- [ ] Lead management module

## Quick Start

### Landing Page
Open `public/index.html` in a browser to preview.

### Development
```bash
# Install Laravel dependencies
composer install

# Run migrations
php artisan migrate

# Seed demo data
php artisan db:seed
```

## Tech Stack
- Laravel 11
- Vue.js 3
- Inertia.js
- Laravel Reverb (real-time)
- Stripe (payments)

## Pricing
- Starter: $29/mo (1 location, 5 coaches)
- Pro: $59/mo (3 locations, 15 coaches)
- Business: $99/mo (unlimited)
